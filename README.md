# CalmKids Academy

> Financial dashboard platform for early childhood education businesses.
> Built for parents of children aged 2-8 who want safe, educational screen time without hyper-stimulation, hidden fees, or frustrating ux..

## Overview

CalmKids Academy is a real-time financial intelligence platform designed specifically for the early childhood education industry. It connects your accounting software, CRM, and operations data into one unified dashboard.

## Features

- **Real-Time Dashboard** — Live financial KPIs specific to early childhood education
- **AI CFO Advisor** — Ask questions and get industry-specific financial guidance
- **QuickBooks Integration** — Automatic two-way sync with your books
- **Industry Reports** — Pre-built reports tailored for parents of children aged 2-8 who want safe, educational screen time without hyper-stimulation, hidden fees, or frustrating ux.
- **Stripe Billing** — Subscription management built in

## Getting Started

1. Clone this repo
2. Copy `.env.example` to `.env.local` and fill in your keys
3. Run `npm install`
4. Run `npm run dev`
